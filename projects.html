Quatro: the Rebrand

Created: August 2023

Collaborators:

Jesica Ohene-Lead Front end Developer

Jesica Ohene-Lead Front end Developer

Sandra Neiras-Lead Designer

It was agreed upon the four brand colours would be used to represent the four fruits that composed the original flavour (combination of pineapple, passionfruit, orange, and grapefruit). We then decided that these flavours would be split individually so that there would be variety

With the use of the css5 grid and media queries that site was able to become compatible with several device sizes including mobile, tablet and desktop
